搭建hexo博客小结

发表于 2019-08-20

今天鼓捣了一个hexo博客，总结一下相关的步骤，以供参考吧
全程使用powershell执行命令

一、部署hexo

下载git

因为要托管到github上，需要git环境

下载node

在windows本地部署hexo需要node环境，所以首先要安装node.js
安装后使用node -v查看node.js版本
使用npm -v查看npm包管理器版本

下载hexo

执行npm install -g hexo-cli下载安装hexo
使用hexo -v查看hexo版本

二、生成博客

新建一个文件夹并切换到此文件夹目录下
执行hexo init初始化博客
执行hexo generate或hexo g生成博客
执行hexo server或hexo s启动服务访问相应地址即可

三、推送到github托管博客

执行npm install hexo-deployer-git --save下载部署hexo到github上的插件
修改博客配置文件_config.yml中depoly部分

执行hexo d

四、上传文章

执行hexo new "新文章"创建文章
文章的位置在source\_posts下

五、更换主题

更换主题需要下载至themes目录下
